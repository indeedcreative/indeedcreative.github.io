In computer networking, port forwarding or port mapping is an application of network address translation (NAT) that redirects a communication request from one address and port number combination to another while the packets are traversing a network gateway, such as a router or firewall.

DNS

https://en.wikipedia.org/wiki/Dynamic_DNS

DNS allows you to dynamically update a domain name to your router's IP address.

So instead of remembering IP address, we can access the information by domain

By setting DDNS : this will always point to the current IP address of my router and therefore will always be able to access my application by using the specific domain name
